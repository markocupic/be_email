# Contao Backend E-Mail
Fügen Sie dem Contao Backend ein E-Mail Plugin hinzu. In den Contao Einstellungen kann definiert werden, woher das Adressbuch seine Adressen zieht. Entweder tl_member oder tl_user oder aus beiden Tabellen.

## Kompatibilität
Die Version 3.x ist nur zu Contao >=4.4 kompatibel.
